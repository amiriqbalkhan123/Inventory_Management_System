# Inventory Management System (Python GUI + MySQL)

A complete **Inventory Management System** built with **Python**, featuring a graphical interface using **Tkinter**, **PyQt5**, and **Kivy**, and data reporting using **Matplotlib**.  
The system is fully connected to a **MySQL backend** and includes modules for managing categories, suppliers, products, customers, employees, transactions, and sales reports.
